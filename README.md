Title: Telco Customer Churn Analysis & Retention Strategy
 Project Overview
This project analyzes a dataset of 7,043 California telecommunications customers to identify the root causes of churn. By exploring demographics, service usage, and financial metrics, I developed data-driven recommendations to improve customer retention.

 Key Insights

High-Value Loss: Churned customers had a significantly higher average monthly charge (~$74) compared to retained customers (~$61).

The "Critical Zone": Tenure analysis revealed a massive churn spike in the first 0–6 months of service.

Contract Risk: Month-to-month subscribers are 18x more likely to churn than those on 2-year contracts.

Top Churn Driver: Competitive pressure (better offers/devices) and support quality were the primary reasons for departure.

 Technologies Used

Python (Pandas, NumPy)

Data Visualization (Matplotlib, Seaborn)

Jupyter Notebook

 Recommendations

Target high-paying month-to-month customers with incentives to switch to long-term contracts.

Implement a "New Customer Success" program for the first 90 days to reduce early-stage churn.

Address service quality issues highlighted in "Churn Reasons" to compete with aggressive market rivals.
